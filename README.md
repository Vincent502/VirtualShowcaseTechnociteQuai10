# VirtualShowcaseTechnociteQuai10

Création de l'espace virtuelle des locaux Technocité au Quai10

## globalité du final
Le projet sera principalement en “Grey Box” et asset du Fab car nous n’avons malheureusement pas d’artiste sous la main.
Lors du lancement de l’application nous seront invité à monter dans l’ascenseur (Pour laisser le temps a l’application de charger les assets + map).

## Start de l'application
Quand la porte de l’ascenseur se réouvre nous somme accueillit par un “Majordome” qui commencera la visite guidée.

## Que va-t-il ce passer dans l'application
Nous serons invité dans différente salle comme la cuisine, l’espace détente, le front desk mais aussi les salles de cours.


## Quoi s'attendre dans l'environement
Nous pourrons remarquer dans chaque salle des personnages animés ainsi que des éléments interactifs.

## Quel fonctionnalité intégrer
Pour nous déplacer dans tout cette espace, nous aurons le système de téléportation.

## Accompagnement visuel 
Au niveau de l'interface utilisateur, nous aurons quelques éléments qui vont suivre le joueur, mais aussi quelques éléments qui seront fixe dans les différentes salles.
